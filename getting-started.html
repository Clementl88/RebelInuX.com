Base Chain Wallet Setup

Setting up MetaMask for Base chain

MetaMask Wallet

The most popular Ethereum/BASE wallet for browsers and mobile

Download MetaMask

Step 1: Install Extension

Visit the official MetaMask website and install the extension for your browser.

Important: Only download from the official website - never from third-party sources.

Step 2: Create New Wallet

Click "Create a new wallet" and follow the setup wizard.

  • Create a strong password (12+ characters)
  • SAVE YOUR SECRET RECOVERY PHRASE
  • Never share your recovery phrase with anyone

Step 3: Add Base Network

Base network is required for $rebelinux. Add it manually:

Manual Configuration:
Network Name: Base Mainnet
RPC URL: https://mainnet.base.org
Chain ID: 8453
Currency Symbol: ETH
Block Explorer: https://basescan.org

Step 4: Add $rebelinux Token

After switching to Base network, add the $rebelinux token:

Contract Address: 0xf95beeF6439ec38fA757238Cdec8417ABDA536bd
How to add:
  1. Click "Import tokens" in MetaMask
  2. Switch to "Custom token" tab
  3. Paste the contract address above
  4. Token symbol should auto-fill as "rebelinux"
  5. Click "Add custom token"

Solana Wallet Setup

Setting up Phantom for Solana chain

Phantom Wallet

The leading Solana wallet for $REBL rewards and trading

Download Phantom

Step 1: Install Extension

Download Phantom from their official website for your browser.

Note: Phantom is also available as a mobile app for iOS and Android.

Step 2: Create New Wallet

Click "Create new wallet" and secure your secret recovery phrase.

  • Write down all 12 words in exact order
  • Store in multiple secure locations
  • Never take screenshots or store digitally

Step 3: Add $REBL Token

Add the $REBL governance token to track your rewards:

Contract Address: F4gh7VNjtp69gKv3JVhFFtXTD4NBbHfbEq5zdiBJpump
How to add in Phantom:
  1. Click the "+" button in your tokens list
  2. Select "Add token"
  3. Paste the contract address above
  4. Token should appear as "$REBL"
  5. Click "Add" to complete

Step 4: Get SOL for Gas Fees

You need a small amount of SOL for transaction fees on Solana:

Buy on Exchange

Purchase SOL on an exchange and withdraw to your Phantom address

Use Jupiter

Swap other tokens for SOL directly in Phantom using Jupiter DEX aggregator

Testnet Faucet

For testing only: Get free test SOL from Solana faucets

Tip: 0.1 SOL is more than enough for thousands of transactions

Security Best Practices

Protect your assets and stay safe

Secret Recovery Phrase

NEVER share your 12-word recovery phrase with anyone. Not even support staff.

  • Write it on paper
  • Store in fireproof safe
  • Never store digitally

Wallet Security

Enable all available security features in your wallets:

  • Use strong passwords
  • Enable biometric authentication
  • Set up transaction confirmations

Scam Prevention

Be vigilant against common crypto scams:

  • Never click suspicious links
  • Verify contract addresses
  • Double-check transaction details

Backup Strategy

Always have multiple backups:

  • Multiple paper copies
  • Store in different locations
  • Test recovery process
